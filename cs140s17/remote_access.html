Guide to remotely accessing LNG computers

Your file system is accessible online here

Login using your PODS ID and passwords and select the PODS domain.

In the top right group, click My H Drive to download and upload your files.

It may ask you to login again, enter PODS for the Domain.

Checkmark the box next to the files you would like to download and then download the zip file that will contains your selected files.

Below the link to your H drive, you can SSH to "Bingsuns" to get a remote terminal.

Since you are remotely accessing the system from the command line, you will only be able to use the command line and as such won't be able to open a gedit window or run other standalone applications like Eclipse. Thus, the command gedit myFile.txt & or even gedit myFile.txt will give an error since an application window can't open. This is a motivating factor for using in-terminal editing programs like Vim, Emacs, and Nano, which would allow you to edit from inside the terminal. You could also edit the files locally and transfer them manually through your H drive.

Working locally with Windows

So you want to work on your Windows laptop? This should help you to get up and running on a Windows environment to work on coursework

There are a few options you can choose between to work on assignments for this course. What you really want to be able to do is edit code, run code, and use git to track and submit your work.

Option 1: Remotely access the LNG computers

See my page on your options for this

This option has the benefit of using an already configured environment that has everything you need. However, everything will be remote which introduces some hoops to jump through if you want to edit files locally (that is, without an in-terminal editor)

Installation: See my page to install PuTTY

Option 2: Emulate a Linux environment via Cygwin

Cygwin is an application for Windows so users can "Get that Linux feeling on Windows". The application consists of a Linux-like terminal with many Linux commands and programs ready to go out of the box. You can access your Windows file structure with cd /cygdrive/c which will put you as if you were in your C drive. This option allows you to use the same command set as you would in the lab as well as being able to edit files locally. That being said, no tool is perfect and cygwin does have some quirks from its nature of being an emulator and not the real deal.

Installation: Download here. Once downloaded, launch the installation wizard. Eventually you will get to a screen that will ask you to check those programs you want to install on top of cygwin. You will want to include Java and Git for now. You can always go back to add others.

Option 3: Use a Windows command prompt

Windows has a terminal that you can use, with similar command functionality, albeit under different names. There is also Windows PowerShell which is slightly more user-friendly if you'd like to learn and use that instead. Powershell is shipped with Windows >=8. This option requires you to download Java 8 and Git for Windows as well as learn the Windows terminal alongside that of Linux. Once these items are installed, you're workflow should be fairly similar to that of in the lab.

Installation:

  • Download Java 8
    • Follow Professor Lander's directions
    • Once correctly setup, go into the terminal and enter commands java -version and javac -version. Make sure you see version numbers of the form 1.8.X_XX for both commands.
  • Download Git for Windows here
  • [OPTIONAL] Download Sublime and/or Notepad++ text editors.

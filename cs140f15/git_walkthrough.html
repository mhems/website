Brief guide to using Git for this course

Note:

Command line usage information typically follows the convention that optional arguments are enclosed in square brackets ( [] ). For instance, you can either run git status to ask git about all your files or run git status file1.txt to ask git about the status of file1.txt

To ask git for help:

git help [command]

To ask git about the status of your files:

git status [files]

To clone a new repository to your local computer:

git clone url

To pull any changes made to the repository to your local computer:

git pull [files]

To add any files you may have changed to be committed:

git add file1 [file2...]

To commit any files you have added:

git commit -m 'a descriptive message' [files]

To push your commited changes:

git push origin master
